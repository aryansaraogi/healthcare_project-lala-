About my Healthcare System

This project is a smart healthcare platform designed to help users understand their health using simple inputs like symptoms, food images, and body measurements. It uses logical models and AI-based concepts to generate useful health insights.

🚀 Features

🩺 Health Checkup

Analyze symptoms and predict possible diseases with risk level and suggestions.

🍎 Diet Analyzer

Upload food images and get nutrition details like protein, carbs, fiber, and calories.

⚖️ Body Check (BMI)

Check your BMI using height and weight and get personalized suggestions.

⚙️ How It Works

  • User enters data (symptoms, food, body details)
  • System analyzes inputs using logic
  • Predictions and suggestions are generated
  • User gets easy-to-understand results

💻 Technologies Used

  • Python (Flask)
  • HTML, CSS
  • Basic AI Logic

🎯 Purpose

The main goal of this project is to make health awareness simple and accessible. It helps users take early steps towards better health by understanding their body.

📖 User Guide

This guide will help you understand how to use each feature of the Healthcare System step by step.

🩺 Health Checkup Guide

👉 Enter your age

👉 Select symptoms (like fever, headache, etc.)

👉 Choose duration of symptoms

👉 Click Analyze

👉 Get disease prediction, risk level & suggestions

🍎 Diet Analyzer Guide

👉 Upload food image

👉 Select food type from dropdown

👉 Click Analyze Food

👉 Get nutrition details (protein, carbs, fiber, calories)

👉 Get diet suggestions

⚖️ Body Check (BMI) Guide

👉 Enter your height (in cm)

👉 Enter your weight (in kg)

👉 Click Check BMI

👉 Get BMI value + health category

👉 Receive fitness suggestions

💡 Tips for Best Results

  • ✔ Enter correct data for accurate results
  • ✔ Select appropriate symptoms
  • ✔ Choose correct food type
  • ✔ Use this system for awareness, not medical diagnosis

⚠️ Important Note

This system provides predictions based on logic and AI concepts. It is not a substitute for professional medical advice. Always consult a doctor for serious health conditions.
